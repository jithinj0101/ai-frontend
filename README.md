AI Frontend - 
================================

This repository contains the React-based frontend for the AI-driven  application.
The application provides an interface for uploading datasets, sending queries, and viewing insights generated by the backend API.

----------------------------------------
Features
----------------------------------------
- React 18+ frontend application
- Upload Excel/CSV files
- Connects to FastAPI backend
- Displays results, insights, and visualizations
- Clean and modular component structure

----------------------------------------
Requirements
----------------------------------------
- Node.js 18+
- npm 7+
- A running backend (FastAPI API)

Backend URL example:
http://localhost:8004

----------------------------------------
Installation & Setup
----------------------------------------

1. Clone the Repository
-----------------------
git clone https://github.com/jithinj0101/ai-frontend.git
cd ai-frontend

2. Install Dependencies (node_modules not included)
---------------------------------------------------
npm install

3. Start the React App
-----------------------
npm start

The application will open at:
http://localhost:3000



----------------------------------------
Backend Integration
----------------------------------------
Ensure the backend (FastAPI) is running:
http://localhost:8004


----------------------------------------
Build for Production
----------------------------------------
npm run build

Output will be in:
build/

# Start the development server
npm start
